Descrição do Projeto


Projeto da cadeira IF762 - Tópicos Avançados em Interface (Visualização de Dados)

Projeto Biciflow, visualizações em Recife.

O projeto visa criar uma aplicação para o entendimento de diferentes dados sobre ciclistas no trânsito do Recife, fazendo o uso de diversas ferramentas de visualização.

Diferentes bases disponiblizadas pelo portal de dados abertos da Prefeitura do Recife foram usadas no projeto, além da utilização das ferramentas D3.js, dc.js e Crossfilter.
